## Developing:

```
npm install -g esy
git clone <this-repo>
esy install
esy build
```

## Running Binary:

After building the project, you can run the main binary that is produced.

```
esy x ReasonGuestLoginOidcClientApp.exe 
```

## Running Tests:

```
# Runs the "test" command in `package.json`.
esy test
```
